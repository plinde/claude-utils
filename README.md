# claude-utils

Tools for humans.

## Tools

| Tool | Description |
|------|-------------|
| [`ccpm`](ccpm/) | Claude Code Plugin Manager - discover, update, and manage plugin marketplaces |
| [`ccss`](claude-code-session-search/) | Claude Code Session Search - browse and search sessions from `~/.claude/projects` |
| [`claude-resume`](claude-resume/) | Resume a Claude Code session by ID |

## Installation

```bash
make install    # Symlinks tools to ~/bin
```

## Usage

```bash
make help       # Show available targets
just help       # Alternative using just
```
